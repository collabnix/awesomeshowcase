## Getting Started

## Clone the repository

```
 git clone https://github.com/collabnix/awesomeshowcase
```

### Prerequisites

- Install Node.js

### Local Development

- Run yarn install in your terminal. 

```
 yarn install
```

- If you want to also install the external docs use:

```
yarn cli start
```

and 

```
yarn cli checkout
```

- Start the service

```
npx docusaurus start
```

Open http://localhost:3000
